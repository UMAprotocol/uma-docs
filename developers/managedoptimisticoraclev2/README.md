# ManagedOptimisticOracleV2

The [ManagedOptimisticOracleV2](https://github.com/UMAprotocol/managed-oracle/blob/master/src/optimistic-oracle-v2/implementation/ManagedOptimisticOracleV2.sol) contract is an updated version of the OptimisticOracleV2 that is permissioned to specific requesters and allows for requests to whitelist who can propose them.&#x20;
